# Interview Performance Analysis System

## 📌 Description
This project analyzes candidate interview performance using Machine Learning.

## 🚀 Features
- Performance Prediction
- KMeans Clustering
- RandomForest Classification
- Data Visualization
- Feedback System

## 🛠️ Technologies Used
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run main.py
